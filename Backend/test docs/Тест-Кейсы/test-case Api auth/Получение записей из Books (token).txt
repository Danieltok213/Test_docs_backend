Название: Получение записей из таблицы Books (корректные данные в запросе с токеном авторизации)

Номер: GB-001

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: GET.
3.В поле URL ввести: http://localhost:8080/admin/books.
4.В хедерах запроса добавить следующий заголовок:
Authorization: token
5.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа должен быть 200 (OK).
Тело ответа должно содержать JSON-объект с записями из таблицы Books.



Название: Получение записей из таблицы Books (корректный запрос без токена авторизации)

Номер: GB-002

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: GET.
3.В поле URL ввести: http://localhost:8080/admin/books.
4.В хедерах запроса не передавать токен авторизации.
5.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 500
Тело ответа должно быть пустым



Название: Получение записей из таблицы Books (корректный запрос с некорректным  токеном авторизации)

Номер: GB-003

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: GET.
3.В поле URL ввести: http://localhost:8080/admin/books.
4.В хедерах запроса добавить следующий заголовок:
Authorization: "неккоректный token" например (Wmdsdkcsdmcfvsckvsj)
5.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 400 (Bad Request).
Тело ответа должно содержать сообщение об ошибке авторизации.

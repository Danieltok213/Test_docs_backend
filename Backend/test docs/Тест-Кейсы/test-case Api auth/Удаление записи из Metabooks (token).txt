Название: Удаление записи из таблицы Metabooks (корректные данные в запросе с токеном авторизации)

Номер: DM-001

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: DELETE.
3.В поле URL ввести: http://localhost:8080/admin/deleteMetabook/1 (где "1" - идентификатор записи для удаления).
4.В хедерах запроса добавить следующий заголовок:
Authorization: token
5.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа должен быть 200 (OK).
Тело ответа должно возвращать id, что указывает на успешное удаление записи из БД.

Номер: DM-002

Шаги:

Название: Удаление записи из таблицы Metabooks (корректный запрос с некорректным токеном авторизации)

Номер: DM-003

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: DELETE.
3.В поле URL ввести: http://localhost:8080/admin/deleteMetabook/1 (где "1" - идентификатор записи для удаления).
4.В хедерах запроса добавить следующий заголовок:
Authorization: "неккоректный token" например (Wmdsdkcsdmcfvsckvsj)

Ожидаемый результат:
Статус код ответа 400 (Bad Request).
Тело ответа должно содержать сообщение об ошибке авторизации.

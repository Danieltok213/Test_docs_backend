Название: Удаление записи из таблицы Books (корректные данные в запросе с токеном авторизации)

Номер: DB-001

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: DELETE.
3.В поле URL ввести: http://localhost:8080/admin/deleteBook/1 (где "1" - идентификатор записи для удаления).
4.В хедерах запроса добавить следующий заголовок:
Authorization: token
5.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 200
Тело ответа должно возвращать id, что указывает на успешное удаление записи из БД.



Название: Удаление записи из таблицы Books (корректный запрос без токена авторизации)

Номер: DB-002

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: DELETE.
3.В поле URL ввести: http://localhost:8080/admin/deleteBook/1 (где "1" - идентификатор записи для удаления).
4.В хедерах запроса не передавать токен авторизации.
5.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 500
Тело ответа должно быть пустым


Название: Удаление записи из таблицы Books (корректный запрос с некорректным токеном авторизации)

Номер: DB-003

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: DELETE.
3.В поле URL ввести: http://localhost:8080/admin/deleteBook/1 (где "1" - идентификатор записи для удаления).
4.В хедерах запроса добавить следующий заголовок:
Authorization: "неккоректный token" например (Wmdsdkcsdmcfvsckvsj)
5.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 400 (Bad Request).
Тело ответа должно содержать сообщение об ошибке авторизации.

Название: Удаление записи из таблицы Creators (корректные данные в запросе с токеном авторизации)

Номер: DC-001

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: DELETE.
3.В поле URL ввести: http://localhost:8080/admin/deleteCreator/4 (где "4" - идентификатор записи для удаления).
4.В хедерах запроса добавить следующий заголовок:
Authorization: token
5.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа должен быть 200 (OK).
Тело ответа должно возвращать id, что указывает на успешное удаление записи из БД.



Название: Удаление записи из таблицы Creators (корректный запрос без токена авторизации)

Номер: DC-002

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: DELETE.
3.В поле URL ввести: http://localhost:8080/admin/deleteCreator/4 (где "4" - идентификатор записи для удаления).

Статус код ответа 500
Тело ответа должно быть пустым

Номер: DC-003

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: DELETE.
3.В поле URL ввести: http://localhost:8080/admin/deleteCreator/4 (где "4" - идентификатор записи для удаления).
4.В хедерах запроса добавить следующий заголовок:

Ожидаемый результат:
Статус код ответа 400 (Bad Request).

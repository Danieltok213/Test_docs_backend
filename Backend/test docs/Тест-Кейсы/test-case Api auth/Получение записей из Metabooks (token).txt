Название: Получение записей из таблицы Metabooks (корректные данные в запросе с токеном авторизации)

Номер: GM-001

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: GET.
3.В поле URL ввести: http://localhost:8080/admin/metabooks.
4.В хедерах запроса добавить следующий заголовок:
Authorization: token
5.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа должен быть 200 (OK).
Тело ответа должно содержать JSON-объект с записями из таблицы Metabooks.

Номер: GM-002

2.Выбрать метод запроса: GET.
3.В поле URL ввести: http://localhost:8080/admin/metabooks.

Название: Авторизация пользователя (корректные данные)

Номер: AU-001

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: POST.
3.В поле URL ввести: http://localhost:8080/auth.
4.В разделе "Body" выбрать опцию "raw" и установить формат "JSON (application/json)".
5.Вставить следующий JSON-запрос в поле Body:
  {
    "login": "login",
    "password": "1234"
  }
6.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 200.
Тело ответа должно содержать JSON-объект пользователя с данными авторизации
  {
    "id": 0,
    "login": "login",
    "password": "",
    "token": "vawMfttzYmXgxSy...",
    "role": "admin",
    "ts": "2023-09-12T14:27:43.955622700"
  }

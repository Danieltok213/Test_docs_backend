Название: Получение полной информации о метакниге через API (позитивный тест)

Номер: A-001

Шаги:
1. Открыть Postman.
2. Выбрать метод запроса: Get.
3. В поле URL ввести: http://localhost:8080/metabookF2/1
4. Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 200 Ok
В теле ответа возвращает полную информацию о конкретной метакниге {json}.

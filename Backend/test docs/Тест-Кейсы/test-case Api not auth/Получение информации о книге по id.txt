Название: Получение информации о книге по id через API (позитивный тест)

Номер: A-001

Шаги:
1. Открыть Postman.
2. Выбрать метод запроса: Get.
3. В поле URL ввести: http://localhost:8080/book/1 
4. Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 200 Ok
В теле ответа возвращается информация о книге под id = 1 {json}.
